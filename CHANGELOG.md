## [0.1.0] - 2025-06-11

### 阶段一：快速 Demo 🎉

首个可运行版本，桌面小猫诞生！

#### 新增

- 基于 Electron 的桌面宠物框架，跨平台支持 macOS / Windows
- 透明、无边框、始终置顶的桌面窗口，支持点击穿透（不遮挡正常操作）
- 智能鼠标穿透：仅当鼠标位于小猫身上时捕获事件，其余区域点击穿透到下层应用
- 状态机驱动的行为系统，行为模块化，便于后续扩展：
  - `idle` 待机：原地停留 + 呼吸感动画
  - `walk` 走动：在桌面随机溜达
  - `follow` 跟随鼠标：鼠标靠近时朝光标方向移动并转身看向光标
- 鼠标拖动：可按住小猫拖到任意位置
- 形象系统：内置 SVG 橘猫占位形象；放入 `src/assets/cat-idle.png` 写实照片可自动替换
- 配置驱动：`config/cat.config.json` 集中管理外观、行为参数，为后续定制化预留性格/癖好字段
- README 项目文档与路线图
