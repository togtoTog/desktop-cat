# 更新日志

本项目所有重要变更都会记录在此文件中。

格式参考 [Keep a Changelog](https://keepachangelog.com/zh-CN/)，版本遵循语义化版本。

## [0.2.0] - 2025-06-11

### 重大升级：2D → 3D 骨骼小猫 🐱✨

小猫从静态 2D 图片升级为可自由活动的 3D 模型！

#### 新增

- 引入 Three.js（WebGL）3D 渲染层，渲染到透明画布，与置顶/穿透窗口完美兼容
- 程序化生成带「骨骼/部件」层级的低多边形 3D 小猫：头、四肢、双眼、双耳、多段尾巴均可独立活动
- 骨骼驱动的基础动作：
  - 待机呼吸（身体起伏）
  - 自然眨眼（随机间隔）
  - 转头看向鼠标（头部独立偏转跟随光标）
  - 走动踏步（四肢交替摆动 + 身体颠簸）
  - 尾巴自然摆动
  - 左右转身（朝向移动方向）
- 2D / 3D 渲染模式可通过配置 `appearance.renderMode` 切换（`"2d"` / `"3d"`），2D 版本完整保留
- 配置新增 `appearance.bodyColor` / `eyeColor`，可调整 3D 小猫的体色与眼色
- 主进程转发渲染进程控制台日志，便于开发调试

#### 变更

- 默认渲染模式为 3D，默认显示尺寸调整为 160px 以便看清模型
- `Cat` 核心类重构：视觉层抽象为 2D/3D 双渲染器，行为状态机逻辑完全复用

## [0.1.0] - 2025-06-11

### 阶段一：快速 Demo 🎉

首个可运行版本，桌面小猫诞生！

#### 新增

- 基于 Electron 的桌面宠物框架，跨平台支持 macOS / Windows
- 透明、无边框、始终置顶的桌面窗口，支持点击穿透（不遮挡正常操作）
- 智能鼠标穿透：仅当鼠标位于小猫身上时捕获事件，其余区域点击穿透到下层应用
- 状态机驱动的行为系统，行为模块化，便于后续扩展：
  - `idle` 待机：原地停留 + 呼吸感动画
  - `walk` 走动：在桌面随机溜达
  - `follow` 跟随鼠标：鼠标靠近时朝光标方向移动并转身看向光标
- 鼠标拖动：可按住小猫拖到任意位置
- 形象系统：内置 SVG 橘猫占位形象；放入 `src/assets/cat-idle.png` 写实照片可自动替换
- 配置驱动：`config/cat.config.json` 集中管理外观、行为参数，为后续定制化预留性格/癖好字段
- README 项目文档与路线图
